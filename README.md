# portal-index-notes

## 项目简介

本仓库用于归档与发布多个独立的 HTML 页面。这些页面本身不针对任何特定的域名或网站，而是作为一份通用的索引与笔记集合，方便查阅与分享。

仓库内的页面可涵盖工具、参考、备忘等多种用途，适合个人或小团队维护使用。

## 目录说明

```
portal-index-notes/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── README.md       # 本文件
└── LICENSE         # 开源许可证（如有）
```

- `pages/`：每个 HTML 文件为一个独立页面，命名建议使用有意义的英文或拼音，便于识别。
- `assets/`：按需存放公共资源，避免重复引用。

## 页面归档说明

所有页面均以静态 HTML 形式存放，不依赖后端服务或数据库。每个页面保持自包含（self-contained），尽量减少外部依赖，确保在离线或本地环境下也能正常查看。

页面内容可以是：

- 个人笔记与学习记录
- 常用工具与链接索引
- 代码片段与示例演示
- 其他适合以 HTML 形式呈现的信息

## 维护说明

- 新增页面请放入 `pages/` 目录，并确保文件名唯一。
- 如页面引用外部资源（在线字体、库文件等），建议在注释中注明来源或提供备用方案。
- 仓库更新时，请保持目录结构清晰，避免重复或冗余文件。
- 欢迎提交 Pull Request 或 Issue 讨论内容改进。
